How do I create objects?
# SIRIUS CODE
#Object
//or
 #Object{}
//or
 #Object{
}
The Basic available types:
# SIRIUS CODE
#Object
#Array
#SiriusObject
// # is a push() equivalent, you can define typed variables instead:
 myobject:Object
//or
 myarray:Array{}
//or
 hybrid:SiriusObject{
}
Advanced Scripting:
# SIRIUS CODE
myobject:Object {
 myvar = Hello World!
 myarray : Array {
  Hello

  10000
  #Object
  #Object{
   otherarray : Array {
    10 = Testing direct index (array[10])
   }
  }
 }
}
